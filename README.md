# 📚 图书管理系统（BookMS）

基于 Spring Boot + MyBatis + 前端 Ajax 实现的前后端分离式图书管理系统，实现了完整的增删改查功能。

---

## ✨ 功能特性

- **用户登录**：简单的用户认证模块
- **图书管理**：
  - 新增图书（含前后端双重校验）
  - 查询所有图书
  - 根据 ID 查询并修改图书
  - 删除图书
- **统一响应**：封装 `Result` 类统一接口返回格式
- **数据校验**：后端参数校验（价格非空、非负等）

---

## 🛠️ 技术栈

- **后端**：Spring Boot、MyBatis、Maven
- **前端**：HTML、jQuery、Ajax
- **数据库**：MySQL
- **版本控制**：Git + GitHub
